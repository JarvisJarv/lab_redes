# Dependências e comandos de instalação do projeto Trabalho Blockchain

## Pré-requisitos
# Instale estas ferramentas antes de executar os comandos abaixo
Node.js 18 LTS ou superior (Vite 5 requer Node.js ^18.0.0 ou >=20.0.0)
npm 9 ou superior (geralmente instalado junto com o Node.js)

## Passo a passo de instalação
# Execute estes comandos na raiz do repositório
npm install
npm install --prefix server
npm install --prefix client

## Executando o projeto em modo de desenvolvimento
# Sobe backend e frontend simultaneamente
npm run dev
